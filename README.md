# Pet Adoption Notice Board

## Project Overview
This is a Laravel-based web application developed for the Server Side Systems (CW02) module. It serves as a notice board for pet adoptions, allowing users to post and view pets available for adoption.

## Features
- **User Authentication:** Secure registration and login.
- **CRUD Operations:** Users can create, read, update, and delete adoption posts.
- **MVC Architecture:** Follows standard Laravel design patterns.
- **Database:** Uses MySQL to store pet and user information.

## Installation
1. Clone the repo.
2. Run `composer install`.
3. Copy `.env.example` to `.env` and configure database.
4. Run `php artisan migrate`.
5. Serve with `php artisan serve`.
